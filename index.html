Coppermine API

Default (4 albums [thumbnail size which is set in the coppermine settings])

<script src="http://seusite.com.br/galeria/api.php"></script>

Custom albums number (example: 8)

<script src="http://seusite.com.br/galeria/api.php?n=8"></script>

Custom thumbnail size (example: 200x250)

<script src="http://seusite.com.br/galeria/api.php?w=200&h=250"></script>

Use GET params (number: n, width: w, height: h)

<script src="http://seusite.com.br/galeria/api.php?n=16w=200&h=250"></script>
